Class Toggles

    Easily add and remove CSS classes

    ScrollMagic offers a convenient way to add and remove CSS classes to elements at certain scroll positions.
    This is particularly useful for in-page navigation menus.

    For more information check out the documentation on Scene.setClassToggle().
